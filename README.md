# 油价守护者 ⛽

> 智能油价监测助手，让每一次加油都明智

## 📱 产品介绍

油价守护者是一款面向车主的油价监测智能助手APP，帮助车主：
- 实时查询全国各省市油价
- 分析油价走势，把握最佳加油时机
- 查找附近最便宜的加油站
- 记录油耗，计算百公里油耗

## 🚀 快速开始

### 方式一：直接打开（推荐）

直接在浏览器中打开 `index.html` 文件即可使用：

```bash
# 用浏览器打开
open index.html
# 或者使用简单的HTTP服务器
npx serve .
```

### 方式二：开发模式

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

## 📋 核心功能

### 1. 实时油价查询
- 显示全国各省市92#/95#/98#汽油、0号柴油今日价格
- 支持按省份/城市筛选
- 油价更新时间提示

### 2. 油价走势分析
- 近7天/30天油价变化趋势图
- 涨跌原因分析
- 智能建议："现在加"还是"再等等"

### 3. 附近加油站比价
- 基于定位推荐周边加油站
- 按价格排序
- 一键导航到站

### 4. 油耗记录
- 记录加油轨迹
- 计算百公里油耗
- 油耗统计报表

### 5. 智能提醒
- 油价上涨前主动推送
- 下调窗口期提醒

## 🛠️ 技术栈

- **前端**：React 18 + Tailwind CSS
- **图表**：ECharts
- **部署**：静态HTML（单文件）
