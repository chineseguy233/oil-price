> 规划日期：2026-04-19
> 当前版本：V3.0（已完成基础功能）

## 核心原则

**砍到最小，先跑通冷启动。**

- 数据粒度不够（省级均价）的功能，往后放
- 冷启动条件不具备（无用户/无数据）的功能，往后放
- 五一后能做的事，不抢五一前的工期

---

## 三个根本问题及回答

**Q1: 这个产品解决什么用户刚需？**
五一自驾出行，用户想知道「跑这条线路要花多少油钱」。高德/腾讯地图能算路线，但不会告诉用户「途经省份油价不同，加山东油比北京便宜」。这个信息差是独有的。

**Q2: 差异化在哪？**
省间油价差异 + 路线规划 + 油费估算，三者结合的工具目前市场空白。

**Q3: 阶段一能交付什么？**
一个落地页 + 路线油费计算，够用户用，够 SEO 抓取，够验证需求是否真实。

## 功能优先级

### P0 — 五一前必须上线

#### F1: 自驾油费计算器（落地页 + API）

**入口**：`/trip` 独立路由，无 TabBar，单页工具

**页面交互**：

```
┌────────────────────────────────────┐
│  🚗 出发地              📍 目的地   │
│  ┌──────────┐      ┌──────────┐   │
│  │ 北京      │  →   │ 上海      │   │
│  └──────────┘      └──────────┘   │
│                                    │
│  车型油耗：🚗 经济型 1.6L  ▼      │
│  油号：92#  ▼                      │
│                                    │
│  [  计算全程油费  ]                │
└────────────────────────────────────┘

↓

┌────────────────────────────────────┐
│  全程约 1,212 公里                 │
│  途经：河北→山东→江苏→上海        │
│                                    │
│  ┌─ 各省油价基准 ─────────────┐   │
│  │ 河北  92# = 8.27元        │   │
│  │ 山东  92# = 8.31元        │   │
│  │ 江苏  92# = 8.23元        │   │
│  │ 上海  92# = 8.14元  ✓最低│   │
│  └────────────────────────────┘   │
│                                    │
│  预计油费：约 ¥96.5               │
│  （油耗7.5L/百公里 × 12.12百公里）  │
│                                    │
│  五一高速免费！省 ¥480            │
│                                    │
│  [  复制行程  ]  [  导航出发  ]  │
└────────────────────────────────────┘
```

**技术实现**：

1. **路线规划**：高德 Web API `https://restapi.amap.com/v3/direction/driving`
   - origin/destination 传入经纬度
   - 返回路线节点列表（含途经点经纬度）

2. **地点→经纬度**：高德地理编码 API `https://restapi.amap.com/v3/geocode/geo`
   - 用户输入城市名 → 返回 `{lat, lng}`

3. **途经省份判断**：
   - 路线节点经纬度 → 调用 `amapReverseGeocode()` 获取省份
   - 汇总途经省份列表

4. **油费计算**：
   ```
   总里程(km) × (油耗L/百公里) / 100 × 各省油价加权平均
   ```
   - 按里程比例加权各省油价

5. **节假日高速免费判断**：
   - 硬编码主要节假日：五一（5月1-5日）、国庆（10月1-7日）、春节
   - 判断当前日期是否在免费期内

**API 新增**：

```
GET /api/route/oil-cost
  ?from=北京&to=上海
  &oil_type=92
  &fuel_consumption=7.5
返回：
{
  total_km: 1212,
  provinces_crossed: ["北京", "河北", "山东", "江苏", "上海"],
  province_prices: { "河北": 8.27, "山东": 8.31, ... },
  oil_cost: 96.5,
  toll_cost: 480,
  is_free_toll: true,
  free_toll_saving: 480,
  route_waypoints: [...]
}
```

**落地页 SEO**：
- `<title>北京到上海油费计算器_五一自驾多少钱</title>`
- `<meta name="description">输入出发地和目的地，计算自驾油费，途经省份油价对比，五一高速是否免费</meta>`
- URL 参数直接可访问：`/trip?from=北京&to=上海`

### 高德 API 注意事项

当前测试 Key 每日 5000 次上限：
- 路线规划每日上限 5000 次
- 地理编码每日上限 5000 次
- 逆地理编码每日上限 5000 次

**五一期间风险**：如果每天 500 个用户查路线，配额可能在一半用户时耗尽。

**应对方案**：配额预警 + 降级（geocode 失败时用直线估算里程，提示「估算值可能有偏差」）

## 冷启动策略

### 五一前：SEO 落地页引流

1. 上线「自驾油费计算器」落地页
   - 关键词布局：「北京到上海油费」「五一自驾油费计算器」「自驾油费多少钱」
   - 落地页 `/trip` 作为主承接

2. 上线「中国油价红蓝榜」
   - 制作可分享的内容，诱导社交传播
   - 「海南油价为什么最贵？」类型内容在五一期间有传播潜力

3. 主动分发
   - 知乎回答「五一自驾从北京到上海多少钱」类问题，植入工具链接
   - 小红书/贴吧发布「各省油价排行」内容
   - 预计五一期间 5000-10000 UV

### 五一后：积累种子用户

- 聚焦「顺路加油最优路线」做出来后，重点找嘀嘀/快车司机试用
- 不是「找司机用」，而是「司机每次加油顺手上报」作为留存机制
- 建立 20-50 人司机微信群，快速迭代反馈
