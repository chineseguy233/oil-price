# 油价守护者 - 产品需求文档 v3.0

## 1. 项目概述

**项目名称**：油价守护者
**项目类型**：H5 + 小程序双端油价查询应用
**Slogan**：让每一次加油都明智
**技术栈**：React + Vite + Taro + Express.js + Python 爬虫
**数据源**：汽车之家（autohome.com.cn/oil）

## 2. 目标用户

- 私家车主（日常通勤/长途出行）
- 网约车司机（成本敏感）
- 货车司机（油耗成本占比高）

## 3. 核心功能

### 3.1 实时油价查询 ✅ 已完成

- 显示全国各省市 92#/95#/98# 汽油、0号柴油今日价格
- 支持按省份筛选，记住用户偏好（localStorage）
- **启动时自动定位** → 自动匹配并选中用户所在省份
- 油价更新时间 + 数据新鲜度实时显示
- 数据来源：汽车之家，每日定时爬取

### 3.2 油价走势分析 ✅ 已完成

- 近 7 天 / 14 天 / 30 天油价变化趋势图（ECharts）
- **点击数据点显示当日详情**：92/95/98/0 全部油价
- 按省份查看，历史数据持续积累

### 3.3 附近加油站比价 ✅ 已完成

- 基于 GPS 定位或 IP 定位查找周边加油站
- 搜索半径可选：1km / 3km / 5km / 10km
- 按距离最近 / 92# 最便宜 / 95# 最便宜排序
- **导航到站**：点击跳转高德地图 Web / App
- 油站信息：名称、地址、距离、电话（可拨打）、营业时间、24小时标识、评分、品牌
- 油价显示为**该省参考价**，明确标注"实际价格以加油站为准"
- 支持分页加载更多

**定位模式区分**：
- 🛰 GPS 精确定位（绿色标签）
- 🌐 网络定位（橙色标签，精度较低）
- 定位失败时显示具体原因并 fallback 到省份数据

### 3.4 油耗记录 ✅ 已完成

- 记录加油日期、里程、油量、油费
- 自动计算百公里油耗
- 本地存储，数据持久化

### 3.5 油价提醒 ✅ 已完成（需登录）

- 设置省份 + 油号 + 涨价阈值（元/升）
- 触发条件：涨幅超过阈值时通知
- 用户订阅制，可设置多个提醒

### 3.6 用户系统 ✅ 已完成

- 手机号 + 验证码登录
- 密码可选设置
- 登录后可同步油耗记录和提醒设置

## 5. 数据源

### 主数据源：汽车之家 (autohome.com.cn)

- URL: `https://www.autohome.com.cn/oil/`
- 数据格式：JSON（Next.js `__NEXT_DATA__`）
- 更新频率：每日（定时任务 `scheduler.sh`）
- 覆盖：全国 31 个省/市/自治区

### 高德地图 (amap.com)

- 用于：附近加油站搜索（POI）、逆地理编码
- API：`restapi.amap.com/v3/place/around` + `v3/geocode/regeo`
- Key: `6e5eadd96b048804120c4fa2cbad220f`

### IP定位兜底

- `ipapi.co` 免费服务（GPS 失败时 fallback）
- 高德 IP API 作为二级兜底

## 6. API 接口

| 接口 | 方法 | 说明 | 认证 |
|------|------|------|------|
| `/api/oil-prices` | GET | 今日油价（所有省份） | 否 |
| `/api/price-changes` | GET | 油价趋势（按省份，参数 days） | 否 |
| `/api/provinces` | GET | 省份列表 | 否 |
| `/api/health` | GET | 数据新鲜度 + 爬虫状态 | 否 |
| `/api/crawl` | POST | 手动触发爬虫 | 否 |
| `/api/auth/code` | POST | 发送手机验证码 | 否 |
| `/api/auth/login` | POST | 手机号+验证码登录 | 否 |
| `/api/auth/set-password` | POST | 设置密码 | 是 |
| `/api/auth/logout` | POST | 登出 | 是 |
| `/api/user/info` | GET | 用户信息 | 是 |
| `/api/sync/fuel` | GET/POST | 油耗记录同步 | 是 |
| `/api/sync/remind` | GET/POST | 提醒设置同步 | 是 |
| `/api/remind` | GET/POST | 获取/添加油价提醒 | 是 |
| `/api/remind/:id` | DELETE | 删除提醒 | 是 |
| `/api/stations/search` | GET | 按省份搜索加油站（模拟数据） | 否 |
| `/api/stations/nearby` | GET | 附近加油站（预留） | 否 |
| `/api/stations/subscribe` | POST | 订阅加油站 | 是 |

## 7. 页面说明

### 油价页（首页）

- 主油价卡片：大字显示当前省份 + 油号价格
- 数据新鲜度标签（刚刚更新 / Xh前 / ⚠️ Xh前）
- 省份选择器（记住最近访问）
- 油号切换（92/95/98/0）
- 各省油价列表（可下翻浏览）

### 趋势页

- 省份 + 时间范围（7/14/30天）筛选
- ECharts 折线图（多油号趋势）
- **点击数据点**：弹出当日油价详情卡片

### 附近页（加油站比价）

- 搜索模式切换：📍 附近 / 🗺 按省份
- 半径选择（附近模式）：1km / 3km / 5km / 10km
- 定位状态提示（GPS / 网络定位 / 失败原因）
- 排序：距离优先 / 92#最便宜 / 95#最便宜
- 油站卡片：名称、地址、距离、电话、营业时间、品牌、评分
- **导航按钮**：跳转高德地图
- 油价显示参考价（该省油价），标注"实际价格以加油站为准"
- 分页加载更多

### 油耗页

- 加油记录录入（日期/里程/油量/油费）
- 自动计算百公里油耗
- 记录列表展示

### 我的页

- 数据健康状态卡片（正常 / 可能过期）
- 手动更新油价按钮
- 油价提醒设置入口
- 收藏/订阅油站入口
