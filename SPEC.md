# 油价守护者 - 产品需求文档

## 1. 项目概述

**项目名称**：油价守护者  
**项目类型**：跨平台智能助手APP（小程序 + H5）  
**Slogan**：让每一次加油都明智

## 2. 目标用户

- 私家车主
- 网约车司机
- 货车司机

## 3. 核心功能

### 3.1 实时油价查询
- 显示全国各省市92#/95#/98#汽油、0号柴油今日价格
- 支持按省份/城市筛选
- 油价更新时间提示

### 3.2 油价走势分析
- 近7天/30天油价变化趋势图
- 涨跌原因分析
- 智能建议："现在加"还是"再等等"

### 3.3 智能提醒
- 油价上涨前主动推送
- 下调窗口期提醒
- 个性化订阅（关注特定地区/油号）

### 3.4 附近加油站比价
- 基于定位推荐周边加油站
- 按价格排序
- 导航到站

### 3.5 油耗记录
- 记录加油轨迹
- 计算百公里油耗
- 油耗统计报表

## 4. 技术架构

### 4.1 前端
- **框架**：Taro 3.x + React
- **样式**：Tailwind CSS
- **图表**：ECharts
- **存储**：本地Storage

### 4.2 后端
- **框架**：Express.js
- **数据**：SQLite
- **数据源**：爬虫抓取

### 4.3 部署
- 前端：Vercel/静态部署
- 后端：Railway/Render/本地

## 5. UI/UX设计

### 5.1 设计风格
- 简洁专业
- 车主易懂的语言
- 重点突出"现在加还是再等等"
- 省钱技巧提醒

### 5.2 色彩方案
- 主色：#2563EB（蓝色 - 信任）
- 涨：#EF4444（红色）
- 跌：#22C55E（绿色）
- 背景：#F8FAFC

### 5.3 字体
- 主字体：系统默认
- 数字：大字号突出

## 6. 页面结构

```
├── 首页（油价查询）
├── 趋势分析
├── 附近加油
├── 油耗记录
└── 我的（设置）
```

## 7. 数据接口

| 接口 | 说明 |
|------|------|
| GET /api/oil/price | 今日油价 |
| GET /api/oil/trend | 油价趋势 |
| GET /api/oil/history | 历史数据 |
| POST /api/user/subscribe | 订阅提醒 |
| GET /api/station/nearby | 附近加油站 |

## 8. 开发计划

### Phase 1: MVP
- 实时油价查询
- 基础UI框架

### Phase 2: 增强
- 油价趋势图
- 油耗记录

### Phase 3: 完整
- 智能提醒
