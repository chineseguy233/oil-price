# 油价提醒功能 - 后期实现

## 目标
当油价达到历史低点或高点时，主动推送通知给用户，提醒用户加油时机。

## 推送方案
**微信服务号模板消息**

- 用户扫码关注服务号，绑定 openid
- 设置想要通知的省份 + 油号
- 后端定时检查并推送，无需用户手动设定阈值

## 触发逻辑

### 自动判断阈值
系统记录用户常用省 + 油号的历史油价，自动判断：

- **降价提醒**：当前油价 < 近 30 天最低价 → 推送"油价降了，适合加油"
- **涨价提醒**：当前油价 > 近 30 天最高价 → 推送"油价涨了，赶紧加油"

### 监控范围
- 用户常用省（根据用户行为自动标记）
- 最多 + 2 个临近省
- 油号跟随车辆设置

## 数据结构

```json
// 用户提醒配置
{
  "openid": "xxxx",
  "provinces": ["北京", "河北"],
  "oilType": "92",
  "enabled": true
}
```

## 推送模板

### 降价提醒模板
```
{{first.DATA}}
省区：{{keyword1.DATA}}
油号：{{keyword2.DATA}}
当前油价：{{keyword3.DATA}}
近30天最低：{{keyword4.DATA}}
{{remark.DATA}}
```

### 涨价提醒模板
同上，remark 内容改为"油价处于高位，有需要请尽快加油"
