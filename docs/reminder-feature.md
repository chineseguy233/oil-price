# 油价提醒功能

## 实现状态：已完成 ✅

系统自动判断阈值，无需用户手动设定。

## 触发逻辑

系统记录用户常用省 + 油号的历史油价，自动判断：

- **降价提醒**：当前油价 < 近 30 天最低价 → 推送"油价降了，适合加油"
- **涨价提醒**：当前油价 > 近 30 天最高价 → 推送"油价涨了，赶紧加油"

## 监控范围

- 用户设置的主监控省份
- 最多 + 2 个相邻省份（共 3 个）
- 相邻省份根据地理位置自动推断（NEIGHBOR_MAP）

## 推送方案

**微信服务号模板消息**（待接入）

- 用户扫码关注服务号，绑定 openid
- 后端定时检查并推送，无需用户手动设定阈值
