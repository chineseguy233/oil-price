# 油价守护者 - 重构设计方案

> 创建日期：2026-03-20

## 1. 项目概述

- **项目名称**：油价守护者
- **类型**：车主工具App
- **技术栈**：React + Vite
- **目标用户**：私家车主、网约车司机

## 2. UI/UX 设计

### 2.1 设计风格
- **主题**：科技感（渐变、动画）
- **主色**：#0B1426（深蓝）
- **强调色**：#4CAF50（绿）、#F44336（红）
- **背景**：白色 + 浅灰渐变

### 2.2 页面结构

```
├── 登录页
│   ├── 深蓝渐变背景
│   ├── Logo + 标题
│   ├── 手机号输入
│   ├── 验证码输入
│   ├── 协议勾选
│   └── 第三方登录
│
├── 首页（油价）
│   ├── 顶部：城市选择 + 刷新
│   ├── 价格卡片：大字 + 变化标签
│   ├── 油号切换：92/95/98/0#
│   ├── 查看详情入口
│   └── 各省油价列表
│
├── 趋势页
│   ├── Tab：7天/30天/90天
│   ├── 折线图
│   └── 统计：最低/最高/当前
│
├── 出行页
│   ├── Tab：加油站/油耗记录
│   ├── 加油站列表（定位+导航）
│   └── 油耗统计（百公里+总油量+总花费）
│
└── 我的
    ├── 头像 + 名字
    ├── 消息提醒
    ├── 数据同步
    ├── 设置
    └── 帮助反馈
```

## 4. 技术架构

### 4.1 前端
- React 18
- Vite
- CSS Modules / styled-components

### 4.2 后端
- Express.js
- SQLite

### 4.3 爬虫
- Python requests + BeautifulSoup
- 数据存储：JSON/SQLite
